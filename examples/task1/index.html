Добро пожаловать!

Вам будет представлена некоторая история, состоящая из высказываний. Высказывания записаны в виде математических выражений в окне выражений. При наведении мыши на высказывание или его выражение, подсвечиваются оба элемента пары. На рисунке к истории вы можете найти улики, которые опровергают некоторые высказывания. Когда найдёте такую улику, кликните по ней, затем по выражению, которое она опровергает, после чего отрицание этого выражения будет помещено в окно доказательств. Окно доказательств - это те данные, которые вы предоставляете компьютерному детективу, который, используя данные выражения, сможет разгадать это дело. Чтобы просто переместить выражение в окно доказательств без его опровержения, кликните по нему. Для возврата выражения из окна доказательств, также необходиму кликнуть по нему.
Когда закончите, нажмите кнопку "расследовать" и детектив проведет расследование.
